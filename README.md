# Groww Inventory System

![Build Status](https://github.com/Ashokpoluka/Growwinvest/actions/workflows/deploy.yml/badge.svg)

Professional, enterprise-grade inventory management system with Groww-inspired aesthetics. Built with a 3-tier architecture (React, Express, MySQL) and AI-powered forecasting.

## 🚀 Quick Start
1. Clone the repository.
2. Follow `DEPLOYMENT.md` for local setup.
3. See `GITHUB_SECRETS.md` for cloud configuration.

## ✨ Features
- **AI Forecasting**: Predictive stock depletion using Google Gemini.
- **Financial Intelligence**: Real-time tax and margin tracking.
- **Globalization**: Multi-language support (English/Hindi).
- **Security**: Hardened CSP, RBAC, and Audit Logging.
- **CI/CD**: Automated testing and Docker deployment via GitHub Actions.
